# 🌦️ Weather App

A responsive weather web application that displays real-time weather information for any city using the **OpenWeatherMap API**.
Users can search a city and instantly view temperature, humidity, wind speed, and weather conditions with dynamic icons.

---

## 🚀 Live Demo

## ✨ Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature in Celsius
* 💧 Humidity percentage
* 🌬️ Wind speed
* 🌤️ Dynamic weather icons (Clear, Clouds, Rain, Drizzle, Mist)
* 📱 Fully responsive (Mobile + Desktop)
* ⚡ Fast and lightweight (Vanilla JavaScript)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript (Vanilla JS)** – Logic & API handling
* **OpenWeatherMap API** – Weather data

## 🔑 Setup API Key

1. Create account on https://openweathermap.org
2. Generate a free API key
3. Open `script.js`
4. Replace:

```js
const apiKey = "YOUR_API_KEY";
```

with your API key.

---

## ▶️ Run Locally

1. Download the project
2. Extract files
3. Open `index.html` in your browser
4. Search any city (Delhi, Mumbai, Dhanbad etc.)
